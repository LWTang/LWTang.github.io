文件目录

  • 移动： mv [ ] “source” “destination”
    • -f:force,强制覆盖
    • -i:询问
    • -u:update,source比较新时覆盖
  • 复制： cp [ ] “source” “destination”
  • 重命名： mv “oldname” “newname”
  • 删除： rm [ ] “文件或目录”
    • -r:文件夹内一同删除
